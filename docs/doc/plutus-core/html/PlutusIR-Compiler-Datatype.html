mkDestructorTy :: Type TyName uni a -> Datatype TyName Name uni a -> Type TyName uni a Source #

Make the type of a destructor for a Datatype. mkDestructorTy functor of List List = forall (a :: *) . (List a) -> (functor of List) = forall (a :: *) . (List a) -> (forall (out_List :: *) . (out_List -> (a -> List a -> out_List) -> out_List))

mkScottTy :: ann -> Datatype TyName Name uni ann -> TyName -> Type TyName uni ann Source #

Make the "Scott-encoded" type for a Datatype, with type variables free. mkScottTy Maybe = forall out_Maybe. out_Maybe -> (a -> out_Maybe) -> out_Maybe
